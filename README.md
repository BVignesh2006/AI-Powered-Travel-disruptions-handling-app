## 🛠️ Tech Stack & Architecture

- **Core**: Python / Flask (Asynchronous Event-Driven)
- **Real-Time Data**: **SSE (Server-Sent Events)** for high-fidelity dash synchronization.
- **Mapping**: **Leaflet.js** with interactive route dashed lines (Green for Rail / Red for Crisis).
- **Automation**: N8N Orchestration / Local Fallback Heuristic Engine.
- **APIs**: 
    - ✈️ **Duffel / Amadeus** (Flight Data)
    - 🚆 **Nilgiri/SNCF Bridge** (Rail Data)
    - 💬 **Twilio / WhatsApp** (Emergency Safety Pulse)
    - ☁️ **OpenWeather** (Predictive Disruption Forecasting)

## 📦 Getting Started & Installation

### 1. Configure Your Sentinel (.env)
Create a `.env` file in the root directory:
```bash
DEEPSEEK_API_KEY=your_key
AMADEUS_API_KEY=your_key
AMADEUS_API_SECRET=your_key
CUSTOMER_PHONE_NUMBER=your_number
EMERGENCY_PHONE_NUMBER=family_number
```

### 2. Launch the Infrastructure
Simply run the portable launcher:
```powershell
./run.bat
```
Visit `http://127.0.0.1:5000` to begin your autonomous journey.
